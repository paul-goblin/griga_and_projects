# griga_and_projects

Use griga to build Grid based Games in Javascript. Some example Games are included in this repo

### Version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Select a Game

Set **currentGameName** in webpack.config.js to the game you want to serve or compile

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

### Build
Compile and build

```sh
$ npm run build
```
